
# Global Development Rules

1. Maintain Lighthouse score above 90.
2. Three.js only allowed in hero sections.
3. All heavy libraries must use dynamic import.
4. Images must use AVIF or WebP.
5. Use shared UI components across apps.
6. Fallback JSON must always exist.
7. Admin routes must require JWT validation.
8. Code must follow SOLID principles.
9. No hardcoded secrets.
10. Maintain clean separation of concerns.
